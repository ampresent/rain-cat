# 🐱🌧️ Rain Cat — 猫猫躲雨奇遇记

一只小猫在雨夜寻找躲雨的地方，遇见温暖的朋友，逐渐探索了解这个世界。

## 🎮 故事

喵喵是一只刚离开家的小橘猫。在一个下雨的夜晚，它蜷缩在湿透的纸箱里……

- 🌧️ **雨夜小巷** — 在寒冷的雨中寻找避风港
- 📚 **温暖书店** — 遇见好心的灰灰，烤火取暖
- 🌺 **秘密花园** — 雨后的花园，蝴蝶和小鸟的世界
- 🌉 **小石桥** — 夕阳下的桥，水中的锦鲤讲述古老的故事
- 🌟 **星空屋顶** — 从高处俯瞰整个世界，和朋友们在一起

## 🐾 角色

| 角色 | 介绍 |
|------|------|
| 🐱 喵喵 | 小橘猫，好奇心强，有点胆小 |
| 📚 灰灰 | 书店老板，一只戴眼镜的老灰猫 |
| 🦋 小蓝 | 蝴蝶，活泼可爱，喜欢水珠 |
| 🐦 知更 | 小鸟，热心肠，歌声好听 |
| 🐟 金金 | 锦鲤，见多识广，说话神秘 |

## 🚀 运行

```bash
# 生成素材
python3 gen_assets.py

# 本地测试
python3 -m http.server 8765
# 打开 http://localhost:8765
```

## 🛠️ 技术

- **引擎**: 基于 LAST SIGNAL 引擎的轻量 point & click 框架
- **素材**: Pollinations.AI 生成 (Flux 模型)
- **特效**: Canvas 雨滴粒子系统
- **无依赖**: 纯 HTML + CSS + JavaScript，无需构建
